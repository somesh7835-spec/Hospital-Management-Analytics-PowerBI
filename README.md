# Hospital-Management-Analytics-PowerBI
End-to-End Hospital Management Analytics Project using Excel, MySQL and Power BI for operational, patient, appointment and revenue analysis.
<h1 align="center">🏥 Hospital Management Analytics Dashboard</h1>

<p align="center">
Excel | MySQL | Power BI
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project focuses on analyzing hospital operations using Excel, MySQL, and Power BI.
The dataset was cleaned and prepared in Excel, analyzed using MySQL queries, and visualized through an interactive Power BI dashboard.
</p>

<hr>

<h2>🛠 Tools & Technologies Used</h2>

<ul>
<li>Microsoft Excel – Data Cleaning & Preparation</li>
<li>MySQL – Data Analysis & Querying</li>
<li>Power BI – Dashboard Development & Visualization</li>
</ul>
